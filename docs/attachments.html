attachments

Syntax 

    dataset attachments COLLECTION_NAME KEY

Description

List the files attached to the JSON record matching the KEY in the collection.

Usage

List all the attachments for k1 in collection “stats.ds”. 

    dataset attachments stats.ds k1

Related topics: attach, detach and prune
