attachments
===========

Syntax
------

```
    dataset attachments COLLECTION_NAME KEY
```

Description
-----------

List the files attached to the JSON record matching the KEY
in the collection.

Usage
-----

List all the attachments for _k1_ in collection "stats.ds".

```shell
    dataset attachments stats.ds k1
```
